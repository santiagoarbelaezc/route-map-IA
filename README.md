<h1 align="center">🗺️ Mapa de Ruta Titanic — CRISP-DM</h1>

<h3 align="center">🚀 Visualización interactiva del proceso de ciencia de datos aplicado al Titanic</h3>

<p align="center">
	Proyecto web que resume y visualiza el flujo completo de un análisis de datos usando el caso Titanic.<br>
	Basado en la metodología CRISP-DM, con fases, herramientas, visualizaciones y métricas.<br>
	HTML/CSS moderno, enfocado en educación y documentación.
</p>

---

## 📋 **Descripción del Proyecto**

Este proyecto presenta un **mapa de ruta interactivo** para el análisis de datos del Titanic, siguiendo la metodología CRISP-DM. Incluye fases, herramientas, visualizaciones y métricas clave, ideal para estudiantes y profesionales de ciencia de datos.

> ⚠️ **Estado del Proyecto:** Finalizado y disponible para consulta

---

## 🏗️ **Estructura Técnica**

- **Frontend:** HTML5, CSS3 (custom, sin frameworks), fuentes Google Fonts
- **Visualizaciones:** Listado de gráficos y métricas utilizadas en cada fase
- **Metodología:** CRISP-DM (6 fases)
- **Interactividad:** Navegación por fases, visualización de matrices y métricas

## 🧩 **Fases CRISP-DM Visualizadas**

| Fase | Descripción | Herramientas |
|------|-------------|--------------|
| **1. Comprensión del Negocio** | Objetivos, contexto, preguntas clave | Documentación |
| **2. Comprensión de los Datos** | Carga, exploración, partición, visualización | Pandas, Matplotlib, Seaborn, Scikit-learn |
| **3. Preparación de los Datos** | Limpieza, encoding, balanceo, estandarización | Pandas, Scikit-learn, Imblearn |
| **4. Modelado** | Entrenamiento, ajuste, funciones, PCA | Scikit-learn, RandomForest, SVC, KNN |
| **5. Evaluación** | Métricas, matriz de confusión, interpretación | Scikit-learn, métricas |
| **6. Despliegue** | Visualización de resultados, alternativas | HTML/CSS, gráficos |

---

## 📊 **Visualizaciones y Métricas**

- Gráficos de barras, pastel, violin, heatmap, pairplot, matriz de correlación
- Métricas: Accuracy, Precision, Recall, F1-Score
- Matriz de confusión visual interactiva
- Listado de herramientas usadas: Pandas, Matplotlib, Seaborn, Scikit-learn, Imblearn

---

## 🏆 **Características del Mapa de Ruta**

- Navegación por fases CRISP-DM
- Descripciones detalladas de cada etapa
- Visualizaciones de datos y modelos
- Métricas explicadas y fórmulas
- Diseño moderno, responsivo y listo para impresión
