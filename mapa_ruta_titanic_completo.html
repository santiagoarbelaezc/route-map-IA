Comprensión del Negocio

El dataset del Titanic contiene información de 891 pasajeros del famoso naufragio. El objetivo es predecir si un pasajero sobrevivió o no (variable sobreviviente: 0 = no sobrevivió, 1 = sobrevivió) a partir de características personales y del viaje. Es un problema de clasificación binaria supervisada.
🎯
Objetivo
Predecir supervivencia de pasajeros (0/1)
📋
Variable Objetivo
Survived / sobreviviente (etiqueta de clase)
📊
Dataset
891 pasajeros × 12 variables
🏷️
Tipo de Problema
Clasificación binaria supervisada
Variables del Dataset Original
📝
Variables Categóricas
Pclass, Name, Sex, Ticket, Embarked, Cabin
🔢
Variables Numéricas
Age, SibSp, Parch, Fare, Survived
Clasificación Binaria Supervisado 891 ejemplos

Evaluación

Métricas de evaluación calculadas con sklearn.metrics para medir el rendimiento de los modelos de clasificación en datos de prueba y mediante validación cruzada.
✅
Accuracy (Exactitud)
accuracy_score(y_true, y_pred, normalize=True) — Proporción de predicciones correctas sobre el total
🎯
Precision
precision_score() — De los predichos como sobrevivientes, cuántos realmente sobrevivieron
📊
Recall (Sensibilidad)
recall_score() — De los que sí sobrevivieron, cuántos fueron detectados correctamente
⚡
F1-Score
f1_score(average='macro') — Balance armónico entre precision y recall
📋
Classification Report
classification_report() — Todas las métricas por clase en un reporte
🔄
Validación Cruzada
KFold(n_splits=10) + cross_val_score() — Media y std del accuracy en 10 folds
Función Auxiliar de Métricas
obtener_performance_classification(y_true, y_pred)
→ return {'accuracy': ..., 'precision': ..., 'recall': ..., 'f1_score': ...}
6

Matriz de Confusión

La matriz de confusión se genera con confusion_matrix(y_test_res, y_pred) de sklearn.metrics. Tiene cuatro celdas que representan los tipos de predicciones. A partir de estas se derivan todas las métricas de evaluación.
✅
Verdaderos Positivos (VP/TP)
Predijo sobrevivió y SÍ sobrevivió
❌
Falsos Positivos (FP)
Predijo sobrevivió pero NO sobrevivió
❌
Falsos Negativos (FN)
Predijo NO sobrevivió pero SÍ sobrevivió
✅
Verdaderos Negativos (VN/TN)
Predijo NO sobrevivió y NO sobrevivió
Derivación de Métricas
Accuracy = (VP + VN) / (VP + VN + FP + FN)
Precision = VP / (VP + FP)
Recall = VP / (VP + FN)
F1-Score = 2 × (Precision × Recall) / (Precision + Recall)
